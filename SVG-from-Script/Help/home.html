SVG from Script Help

SVG from Script was created to help create SVG files for laser cutting projects. Many times, repetative shapes need to be created and I wanted a way to automate the process. This application allows the user to write a script file to generate an SVG file. The script utilizes C# and provides serveral libraries by default.

  • System.Drawing
  • System.Drawing.Drawing2D

The graphics object is available to the script and is instantiated with the graphics variable "G". The following is an example script to draw some circles in an X pattern. 

Pen penGreen = new Pen(Color.Green, 3);
Pen penPink = new Pen(Color.Pink, 2);

G.DrawLine(penGreen, 0, 0, 200, 200); 
G.DrawLine(penGreen, 200, 0, 0, 200);
for(int i=0; i < 8; i++) {
    G.DrawEllipse(penPink, i * 25, i*25, 25, 25);
    G.DrawEllipse(penPink, i * 25, (7*25)-(i*25), 25, 25);
}
